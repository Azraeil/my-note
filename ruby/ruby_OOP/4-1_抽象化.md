# 抽象化
* 專注於本質
* 忽略不相關的部分
* 留下最重要的，最相關的資料

例如說真實世界裏有很多人
抽取出本質，形成"人"的概念

情境A：公司業務

```rb
class 公司業務
  Attributes : 員工編號, 名字, 銷售業績
  Methods : 打電話, 銷售
end
```

情境B：棒球員

```rb
class 棒球員
  Attributes : 姓名, 身高, 體重, 年齡, 隊伍
  Methods : 投球, 打球, 接球, 跑壘
end
```
