## 認識 `資料庫` 與 `關聯式資料庫`
* 什麼是資料庫？
* 什麼是關聯式資料庫？
___

## 什麼是資料庫(Database)？
* Database 是 多個 `Table` 的集合
___
## 什麼是關聯式資料庫(Relational Database)？
* 在關聯式資料庫裡，資料之間有"明確的關係"

舉例：相簿 app, 可以有三個 `table`  
users table  
albums table  
photos table  

### 資料之間有明確的關聯性
users 擁有 albums
albums 裏有 photos
